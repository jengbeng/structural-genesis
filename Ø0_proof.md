• distinctions  
• reactions  
• cycles  
• multiplicity

• has no cycles

This is the definition of **Ø₀**.

## II. Ø₀ — the minimal non-structure

Ø₀ is not emptiness.

It is the **unique state that:**

• contains no distinctions  
• contains no cycles  
• contains no history  
• contains no subjects  
• contains no reaction  
• **but allows distinctions to become possible**

If Ø₀ *forbade* distinctions → structure would be impossible.  
If Ø₀ *contained* distinctions → it would be θ₁, not Ø₀.

Thus:

**Ø₀ is the only possible beginning and end of any structural cycle.**
